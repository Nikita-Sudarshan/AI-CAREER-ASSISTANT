# 💼 AI Career Assistant

An end-to-end Machine Learning web app that analyzes resumes and recommends suitable career paths along with skill gap insights.

---

## 🌐 Live Demo

👉 https://huggingface.co/spaces/YOUR_USERNAME/ai-career-assistant

---

## 🚀 Features

- 📄 Upload resume (PDF) or paste text
- 🎯 Predict Top 3 career roles using NLP embeddings
- 🧠 Extract skills from resume
- ❌ Identify missing skills for target roles
- 📌 Explain why a role is recommended

---

## 🧠 Tech Stack

- **Frontend:** Streamlit
- **ML/NLP:** Sentence Transformers
- **Language:** Python
- **PDF Processing:** PyPDF2

---

## ⚙️ How to Run Locally

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```
